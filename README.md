KasiMarket

KasiMarket is a Community-to-Community (C2C) online marketplace developed as part of an Eduvos software development project. The platform allows users to buy and sell products in a simple, secure, and user-friendly environment.
---

About my Project

KasiMarket was created to provide a digital marketplace where community members can connect to buy and sell products. Users can register, browse products, apply to become sellers, and manage their own listings.
---

Features:

User Features:
- User registration and login
- Secure password hashing
- Browse available products
- View product details
- Apply to become a seller
- User-friendly dashboard

Seller Features:
- Submit seller application
- Add new products
- Edit products
- Delete products
- View and manage "My Products"

Administrator Features:
- Approve or reject seller applications
- Manage users
- Manage marketplace products
- Monitor platform activity

---

Installation
1. Clone the repository:

```bash
git clone https://github.com/mformatthew05-oss/KasiMarket.git
```
2. Move the project into your web server directory (e.g. `htdocs` for XAMPP).
3. Import the MySQL database into phpMyAdmin.
4. Update the database connection in `config.php`.
5. Start Apache and MySQL.
6. Open your browser and navigate to:
http://localhost/KasiMarket
